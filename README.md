# Apps and tools I use on macOS

## Apps

- [Anybox](https://anybox.app/) - bookmarks manager
- [Arc](https://arc.net/) - browser
- [BetterDisplay](https://github.com/waydabber/BetterDisplay) - easily manage external displays
- [CleanShot X](https://cleanshot.com/) - replacement for macOS screenshot/video capture tool
- [Craft](https://www.craft.do/) - note taking
- [FlashSpace](https://github.com/wojciech-kulik/FlashSpace) - replacement for macOS Spaces
- [Ghostty](https://ghostty.org/) - fast, feature-rich, and cross-platform terminal emulator
- [HTTPie](https://httpie.io/) - API testing app
- [Hidden](https://github.com/dwarvesf/hidden) - hide menu bar icons
- [IINA](https://iina.io/) - media player
- [Karabiner-Elements](https://karabiner-elements.pqrs.org/) - keyboard customizer
- [Markdown Preview](https://anybox.ltd/markdown-preview) - quick look extension for Markdown files
- [Mochi](https://mochi.cards/) - flashcards with spaced repetition
- [Notion Calendar](https://www.notion.com/product/calendar) - calendar
- [OK JSON](https://okjson.app/) - JSON formatter
- [OrbStack](https://orbstack.dev/) - replacement for Docker Desktop
- [Raycast](https://www.raycast.com/) - replacement for macOS Spotlight
- [Signal](https://signal.org/) - messaging app
- [Superkey](https://superkey.app/) - keyboard enhancement
- [Swish](https://highlyopinionated.co/swish/) - window manager
- [TablePlus](https://tableplus.com/) - database tool
- [Things](https://culturedcode.com/things/) - personal task manager
- [Zed](https://zed.dev/) - code editor
- [1Password](https://1password.com/) - password manager

## CLI tools

- [atuin](https://github.com/atuinsh/atuin) - shell history
- [bat](https://github.com/sharkdp/bat) - replacement for `cat` with syntax highlighting
- [brew](https://brew.sh/) - package manager
- [carapace](https://github.com/carapace-sh/carapace-bin) - multi-shell completion library
- [delta](https://github.com/dandavison/delta) - syntax-highlighting pager for git, diff, grep, and blame output
- [fd](https://github.com/sharkdp/fd) - replacement for `find` 
- [fzf](https://github.com/junegunn/fzf) - command-line fuzzy finder
- [gitalias](https://github.com/gitalias/gitalias) - huge list of useful aliases for `git`
- [micro](https://github.com/zyedidia/micro) - modern and intuitive terminal-based text editor
- [mise](https://github.com/jdx/mise) - dev tools, env vars, task runner
- [nushell](https://github.com/nushell/nushell) - new type of shell
- [ripgrep](https://github.com/BurntSushi/ripgrep) - replacement for `grep` 
- [starship](https://github.com/starship/starship) - fast and customizable prompt for any shell
- [zoxide](https://github.com/ajeetdsouza/zoxide) - replacement for `cd`
